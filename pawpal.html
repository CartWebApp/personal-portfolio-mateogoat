Overview

PawPal is a built from the ground up pet wellness tracker that allows users to add pets, add veterinarian appointments, log daily medications, daily walks, health trends, and a built-in to-do list for feeding.

Concepts Learned

  • Semantic HTML: I used semantic tags inside my HTML to make sure screen readers are usable on my webpage, including article, section, footer, and more.
  • Responsive Design: I learned how to make my website usable not just on a desktop computer, but also on a mobile phone, which is mainly what a user would interact with my website.
  • CSS Styling: This was my first webpage, which includes all CSS styling required to make it function, including the usage of display: grid, display: flex, and following WCAG guidelines with my color choices.

Problem Solutions

CSS Positioning: I struggled with CSS positioning, specifically display: flex, which had margin errors and the nav bar not working properly. After receiving help from my instructor, I learned how to use translation in CSS to allow for refined positioning and to use margins much more efficiently than before.

Habits of Mind

  • Key Communication: I spoke with my instructor heavily on if there were issues with the CSS styling, or if I had equal work being done with my team. This was an important step in working as a team, as you may need to tell your employer if there is any issues during development.
  • Team Leadership: Holding the weight of this projects proves to not just my peers, but also future employers that I am capable of working more than the bare minimum; most of tis webpage was designed and coded by me, with assistance from my peers alongside my instructor if I struggled on anything.
